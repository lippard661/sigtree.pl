# sigtree.pl
Perl script to do file integrity monitoring for *BSD, Linux, macOS. Optionally uses privilege separation.

This is version 1.22 of 31 January 2026 from https://www.discord.org/lippard/software/

sigtree-1.22.tgz is a Legion of Dynamic Discord signify-signed OpenBSD package. Signify public key is https://www.discord.org/lippard/software/discord.org-2026-pkg.pub

(Last version without dependency on Signify.pm or Parallel::ForkManager is 1.18d.)
